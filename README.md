# plasma6-thememenu
A bash script for Plasma 6 (Fedora KDE) themes. Save your configs and load them at any time.

# How to run 
Put this in your terminal: 
`nano ~/.bashrc`
Scroll to the bottom and pase the contents of `thememenu.bash`
`Ctrl + O`, `Enter`, `Ctrl + X`
Now you can back up your themes :3
